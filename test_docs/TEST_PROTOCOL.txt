Atlas Discovery OS - Phase 1 Test Protocol
============================================
Date: February 2026
Phase: Property Prediction & Toxicity Analysis

SETUP
-----
1. Create a project in Atlas (e.g., "Discovery OS Test")
2. Ingest all .txt files from this folder into that project
3. Wait for ingestion to complete (check file status)
4. Ensure an LLM is loaded (local GGUF or API model)

TEST QUERIES (copy-paste these into the chat)
----------------------------------------------

TEST 1: Basic Property Prediction (Aspirin)
Query: "What are the molecular properties of aspirin? Its SMILES is CC(=O)OC1=CC=CC=C1C(=O)O"
Expected: Agent calls predict_properties -> returns MW~180.16, LogP~1.19, QED~0.55
Expected: Agent calls search_literature -> finds aspirin pharmacology doc
Expected: Agent calls check_toxicity -> returns clean (no alerts)

TEST 2: Property Prediction (Caffeine)
Query: "Predict the properties and check toxicity of caffeine (SMILES: CN1C=NC2=C1C(=O)N(C(=O)N2C)C)"
Expected: MW~194.19, LogP~-0.07, HBD=0, HBA=6, clean toxicity profile

TEST 3: Known Toxic Molecule (Michael Acceptor)
Query: "Check the toxicity profile of acrolein, SMILES: C=CC=O"
Expected: check_toxicity flags michael_acceptor alert
Expected: search_literature retrieves structural alerts reference material

TEST 4: Drug Comparison via Literature
Query: "Compare the drug-likeness of ibuprofen versus aspirin using their molecular properties"
Expected: Agent calls predict_properties for both molecules
Expected: search_literature pulls NSAID comparison and individual drug profiles

TEST 5: Complex Query (Property + Toxicity + Literature)
Query: "Evaluate diclofenac (OC(=O)CC1=CC=CC=C1NC1=C(Cl)C=CC=C1Cl) as a drug candidate. Check its properties, toxicity alerts, and find relevant safety literature."
Expected: predict_properties returns MW~296, LogP~4.5
Expected: check_toxicity may flag aniline-like substructure
Expected: search_literature retrieves NSAID safety comparison data

TEST 6: Lipinski Violator (Paclitaxel)
Query: "What happens when we check properties of paclitaxel? SMILES: CC1=C2C(C(=O)C3(C(CC4C(C3C(C(C2(C)C)(CC1OC(=O)C(C5=CC=CC=C5)NC(=O)C6=CC=CC=C6)O)OC(=O)C7=CC=CC=C7)(CO4)OC(=O)C)O)C)OC(=O)C"
Expected: MW~854, multiple Lipinski violations, but agent should contextualize using literature

TEST 7: Invalid SMILES Handling
Query: "Predict properties for this molecule: INVALID_SMILES_STRING"
Expected: predict_properties returns valid=false, error message
Expected: Agent handles gracefully and explains the error

WHAT TO RECORD
